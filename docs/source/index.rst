欢迎访问 CFMS 开发者手册！
===================================

**CFMS** (Classified File Management System) 是用于分类化管理多密级文档的一整套系统。
本开发手册将系统介绍有关CFMS的相关技术性用法。

CFMS 是自由软件，不做任何保证：在特定的条件下，任何人均可自由修改和再分发本软件，并享有其他多种自由。
请查阅本项目的许可证以获取更多信息。

.. note::

   仍在早期开发阶段。每一个新版本都可能发生大的改动；因此，本文档或不能及时反映最新版本的特性，甚至在某些时候造成误导。
