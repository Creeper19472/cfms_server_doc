在开始之前
======================

关于项目状态
--------------

CFMS (Classified File Management System) 目前处于 **Alpha 开发阶段**，这意味着：

1. **功能尚未完善**：许多计划中的功能仍在开发中，现有功能可能存在缺陷。
2. **API 可能变动**：接口定义、数据结构和协议在不同版本之间可能发生重大变化。
3. **安全性未经审计**：系统尚未经过全面的安全审计，不应在生产环境中处理敏感数据。
4. **稳定性待验证**：系统的稳定性和性能尚未在实际场景中得到充分验证。

安全警告
---------

.. warning::

   **请勿在生产环境中使用 CFMS！**
   
   任何试图将 CFMS 投入生产环境的行为都是不明智的，除非您：
   
   - 已进行全面的安全审计
   - 理解并接受所有潜在风险
   - 有能力及时修复可能出现的安全问题
   - 为潜在的数据丢失或泄露做好了准备

项目背景
---------

CFMS 最初是为视觉小说《溯流而上》的需求而开发的。在开发优先级上，功能实现和用户界面优先于安全性考量。
虽然项目实现了基本的安全措施（如密码哈希、JWT 认证、SSL/TLS 加密等），但这些措施的有效性尚未经过
严格的安全专家审查。

适用人群
---------

本项目适合：

- 学习 WebSocket 服务器开发的开发者
- 研究文档管理系统架构的技术人员
- 参与 Alpha 测试并能接受风险的早期采用者
- 为项目贡献代码和改进的开发者

如果您希望参与测试或贡献代码，我们非常欢迎！请通过 GitHub Issues 提交改进建议和错误报告。

技术要求
---------

在开始使用 CFMS 之前，您应该具备：

- Python 3.11+ 的基础知识
- WebSocket 协议的基本理解
- 数据库（SQLite 或 MySQL）的基本操作能力
- Linux/Unix 系统的基本使用经验（推荐）
- 对信息安全有基本的认识
